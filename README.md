# Мини-система сбора и анализа данных

Цель: создать end-to-end систему для генерации, хранения и анализа данных фитнес-трекера.

## Компоненты системы
- **Генератор данных**: Python-скрипт с 10 пользователями, реалистичным поведением
- **База данных**: PostgreSQL
- **Визуализации**: Redash
- **Анализ**: Jupyter Notebook

## Как запустить
1. Установите Docker и Git
2. Клонируйте репозиторий
3. Выполните:
   ```bash
   docker-compose up --build
